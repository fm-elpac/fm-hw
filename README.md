## 目录

+ **`fmp/`**
  胖蚊子协议

+ **`hw/`**
  硬件设计

  - `pcb_kicad/`
    印刷电路板 (kicad)

  - `3d_freecad/`
    3D 模型 (freecad)

  - `3d_blender/`
    3D 模型 (blender)

  - `chisel/`
    数字电路设计
