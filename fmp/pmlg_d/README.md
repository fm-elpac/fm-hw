# PMLG_D
胖喵学习游戏的 git 仓库数据格式


## 分支名称

+ `main` 主索引

+ `i_`* 索引分支

+ `m_`* 镜像分支

+ `g_`* 游戏分支 (内容项目)

+ `a_`* 资源分支

+ `c_`* 评论分支

索引类型:

+ 群组索引

+ 成员索引

+ 仓库索引

+ 项目索引

+ 镜像索引

+ 资源索引

+ 评论索引


## 根目录文件

每个分支根目录包含以下文件:

+ `LICENSE`

  默认为 `CC BY-SA 4.0`, 可修改.

+ `README.md`

  默认自动生成, 可修改.
  相当于注释.

+ `pmlg.json`

  胖喵学习游戏 根数据文件.
  别的文件都在这个文件中直接/间接引用.


## 支持的文件格式

+ 图文

  - 纯文本 (utf-8)

  - markdown

  - 图片: png, jpg, gif, webp

  - pdf

+ 视频/音频 (ffmpeg)

+ HTML (网页)

+ 代码: js, WASM

+ TODO 3D 模型


## `pmlg.json`

根键:

+ `pmlg_d` 数据文件格式版本

+ `pmlg_v` 客户端软件版本


## 上游仓库 (upstream git)

+ `pmlg_i18n.json` 包含客户端的 i18n 翻译.


## 联系方式

基本上都是可选的.

+ 昵称, 个性签名, 更多资料 (键值对)

+ URL

  - `url_git` PMLG git

+ 数字签名

  - `key_gpg` GPG 公钥 (加密)

+ `web` 公开的网络方式:

  - `site` 网页:
    可以是个人网站, 也可以是平台的个人页面 (比如 b 站)

  - `ddns` (设备 DNS, IPv6 直连, WebRTC)

+ `web_r` 仅接收的网络方式:

  - `email` 邮箱

+ `im` 开放的聊天平台 (第三方公开接口)

  - matrix

+ `im_o` 不开放的聊天平台:

  - QQ

  - 微信

+ `o` 非网络方式:

  - 手机号

  - 地理地址
