# FMP-RA1
胖蚊子遥铠协议

用于 `FMRA-1` 胖蚊子遥铠.

+ 基于 UDP / HTTP/3 ?

  (WebTransport)

+ 力反馈: 最大 64 个自由度 (身体 + 双手)

  每个自由度包括 2 个标量: 位置, 力矩 (旋转关节)

  每个标量使用 IEEE754 单精度浮点数表示 (4 字节)

  每个自由度 8 字节, 所有自由度 512 字节

+ 工作频率: 1kHz

  数据量 512kB/s

+ 网络质量: 丢包率不超过 5%, 抖动不超过 10%

+ 连接异常判定: 连续丢包 3 个 (3ms),
  或最近 50ms 内丢包率超过 10% (累计 5 个)
