## 主要参数

+ 板卡厚度: 1.0mm (4 层板)

+ 金手指 长度: 4.0mm

+ 金手指 间距: 1.0mm, 宽度 0.8mm

+ 双面, 缺口宽度: 2.0mm (4 个金手指)

+ 板卡宽度: 20mm (36 引脚), 24mm (44 引脚)

+ 通用引脚: 18 / 25 个 (E0 ~ E24)

+ 金手指 占用面积: 5 x 20mm (5 x 24mm)

+ 插座 占用面积: 5 x 28mm (5 x 32mm)
