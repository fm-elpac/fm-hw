# FMP-C2
胖蚊子电源连接器


+ `FMP-C2a` 小功率直流电源接口

  基于 2.54mm 排针 (4/5/6/7pin), 2.5A (10 ~ 30W)

  5V, 3.3V (可选), 12V (可选)

+ `FMP-C2b` 中功率直流电源接口

  基于 ATX 电源接头 (4.2mm 双排, 6/8pin), 10A (30 ~ 240W)

  5V, 3.3V (可选), 12V (可选)

+ `FMP-C2c` 大功率直流电源接口

  基于 XT 系列接口: XT30 (15A), XT60 (30A), XT90 等

  5V, 12V


## FMP-C2a

单排针 2.54mm (2.5A), 4/5/6/7pin

+ 4/5pin: 5V (12.5W), 3.3V (8.2W, 可选 NC)

+ 6/7pin: 12V (30W)

针脚定义:

| 序号 | 名称 | 描述 | 备注 |
| :--: | :-- | :-- | :-- |
| 1 | GND | 接地 |  |
| 2 | NC | 缺口 | 用于识别方向 |
| 3 | V5 | 5V 2.5A (12.5W) |  |
| 4 | V3 | 3.3V 2.5A (8.2W) | 可选 |
| 5 | GND | 接地 |  |
| 6 | V12 | 12V 2.5A (30W) | 可选 |
| 7 | GND | 接地 | 可选 |


## FMP-C2b

双排 ATX 4.2mm (10A), 6/8pin

+ 6pin: 5V (50W), 3.3V (30W), 12V (120W)

+ 8pin: 12V (240W)

针脚定义:

| 序号 | 名称 | 描述 | 备注 | 右侧 | 名称 | 描述 | 备注 |
| :--: | :-- | :-- | :-- | :--: | :-- | :-- | :-- |
| 1 | NC | 缺口 | 用于识别方向 | 2 | V3 | 3.3V | |
| 3 | GND | 接地 | | 4 | V5 | 5V | |
| 5 | GND | 接地 | | 6 | V21 | 12V | |
| 7 | GND | 接地 | 可选 | 8 | V22 | 12V | 可选 |

+ 通过 2.54mm 排针连接: 5VSB, PWON


## FMP-C2c

+ FMPP-1 胖蚊子电力线通信协议 使用本接口.
