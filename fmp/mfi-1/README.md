# MFI-1
机器优先界面: 中型低速 4 自由度机械臂末端工具自动连接器


## 主要设计参数

+ 自由度个数: 4 (1 个主自由度 + 3 个辅助自由度)
+ 旋转正方向: 逆时针 (与数学习惯保持一致)

+ 制造方式: 3D 打印 (FDM)
+ 材料: 塑料 (PLA) (TODO ABS/陶瓷)
+ 制造精度: 0.5 ~ 0.2mm

+ 外形: 圆角矩形
+ 颜色: 黄色 (黑色标记)

+ 最高工作温度: 50 摄氏度 (TODO)


## 连接面布局

```
+---------------+
|               |
|  2  xxxxx  1  |
|               |
|  x    0    x  |
|               |
|  3  xxxxx  4  |
|               |
+---------------+
```

+ 原点 (0): 主自由度 (自由度 0)

+ 第一象限 (1): 辅助自由度 (自由度 1)

+ 第二象限 (2): 辅助自由度 (自由度 2)

+ 第三象限 (3): 辅助自由度 (自由度 3)

+ 第四象限 (4): 方向标记 (凸起)

+ `xxx`: 锁定装置

+ (可选) 周围密封圈
